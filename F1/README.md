# F1 - Study Screener (Frontend)

## Prerequisites
- Node.js 18+
- npm

## Install
```bash
npm install
```

## Run (Development)
```bash
npm run dev
```
Then open the URL shown by Vite (usually http://localhost:5173).

## Build for Production
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```

## Notes
- Main app entry: `src/main.jsx`
- Main component: `src/StudyScreener.jsx`
- Mock data: `src/mockStudies.json`
